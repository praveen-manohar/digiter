# Digiter
***Digiter*** - META TAG Generator with Flask, python - Developed by Praveen Manoharan

Live demo, visit visit [Digiter](https://digiter-seo.herokuapp.com/) on your device.


# Start

Make app.py as main file

**export FLASK_APP=app.py**

Run the flask app

**flask run**
